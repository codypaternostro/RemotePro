    Get-RpConnectionProfile

    SYNOPSIS

    Retrieves and displays RemotePro connection profiles or the credentials XML.

    SYNTAX

    ViewProfiles (Default)

    Get-RpConnectionProfile [-Name <String>] [-All] [-ProgressAction <ActionPreference>] [<CommonParameters>]

    ViewXml

    Get-RpConnectionProfile [-ViewXml] [-ProgressAction <ActionPreference>] [<CommonParameters>]

    DESCRIPTION

    The Get-RpConnectionProfile function retrieves connection profiles based on the provided parameters. It can either show all profiles, a specific profile by name, or open the credentials XML file for direct viewing.

    EXAMPLES

    EXAMPLE 1

    Get-RpConnectionProfile -All

    Displays all available RemotePro connection profiles.

    EXAMPLE 2

    Get-RpConnectionProfile -Name "ProfileName"

    Retrieves and displays the connection profile named "ProfileName".

    EXAMPLE 3

    Get-RpConnectionProfile -ViewXml

    Opens the credentials XML file associated with RemotePro connection profiles for direct viewing.

    PARAMETERS

    The name of the connection profile to retrieve. Used when the user wants to view a specific profile. Only valid in the 'ViewProfiles' parameter set.
